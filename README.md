## Description

A Seed to start a Nest application easily with some features already configured

Our Features:

- Prisma Configured
- Docker Compose with PostgresSQL
- Docs with Swagger
- Validation with class validators

Future features:

- Cookies
- Authorization and authentication
- Logging
- Middlewares

## Installation

```bash
$ yarn install
```

## Start database

```bash
$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
