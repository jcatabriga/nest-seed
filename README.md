## Installation

```bash
$ yarn install
```

## Start database

```bash
$ docker-compose up -d
```

## Run Migrations

```bash
# You can edit prisma/schema.prisma before run (optional)
$ npx prisma migrate dev
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
